# 🐓 نظام مراقبة قطيع H&N Super Nick
## مزرعة شديد — مرحلة التربية

---

## 📦 هيكل المشروع

```
flock_app/
├── app.py               # التطبيق الرئيسي (Streamlit)
├── data_loader.py       # تحميل وتنظيف بيانات Excel
├── dashboard.py         # رسوم بيانية ومؤشرات KPI
├── analysis.py          # مؤشر صحة القطيع والتحليل الذكي
├── rules.py             # قواعد نظام الإنذار
├── requirements.txt     # المكتبات المطلوبة
└── Sheded_Farm_Weekly_Report____week__2_.xlsx  # ملف البيانات
```

---

## 🚀 التشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
cd flock_app
streamlit run app.py
```

### 3. فتح المتصفح
افتح: `http://localhost:8501`

---

## 📊 الصفحات

| الصفحة | الوصف |
|--------|-------|
| لوحة التحكم | المؤشرات الرئيسية + رسوم بيانية يومية |
| نظام الإنذار | تنبيهات فورية + سجل التنبيهات |
| التحليل الذكي | مؤشر الصحة + توصيات تصحيحية |
| التقارير الأسبوعية | ملخص أسبوعي محسوب من البيانات اليومية |
| المقارنة بالقياسي | الأداء الفعلي vs معيار H&N Super Nick |

---

## 🔄 تحديث البيانات

- **ارفع الملف** من الشريط الجانبي لتحديث كل شيء تلقائياً
- عند إضافة صفوف جديدة لشيت `Daily Date` وإعادة رفع الملف، تتحدث جميع المؤشرات والرسوم
- فعّل "تحديث تلقائي" للمراقبة المستمرة كل دقيقة

---

## ⚙️ المعايير القياسية المُدمجة

البيانات القياسية لسلالة **H&N Super Nick** مرحلة التربية (Rearing):
- استهلاك العلف اليومي لكل طائر (بالجرام)
- استهلاك المياه اليومي لكل طائر (بالمل)
- نطاقات درجة الحرارة المناسبة
- حدود الرطوبة
- نسبة النفوق المقبولة

---

## 📋 أعمدة ملف Excel المدعومة

| العمود | الوصف |
|--------|-------|
| `عمر القطيع (باليوم)` | عمر القطيع |
| `الحرارة الحالية (مئوية)` | درجة الحرارة |
| `الرطوبة الحالية (%)` | الرطوبة |
| `النافق اليومي` | عدد النافق |
| `استهلاك العلف اليومي (بالكيلو جرام)` | كمية العلف |
| `استهلاك العلف لكل طائر (بالجرام)` | علف/طائر |
| `استهلاك المياه اليومي (باللتر)` | كمية المياه |
| `الملاحظات` | ملاحظات يومية |
| `وجود إصابة مرضية` | حالة الإصابة |
